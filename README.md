# WITCHER

2D-платформер в стиле Ведьмака, написанный на Python с использованием библиотеки Pygame.

## 📖 Описание игры

Вы играете за ведьмака, который сражается с монстрами в болотистой местности. Игра включает в себя боевую систему с комбинациями атак, магические способности, систему здоровья и маны, а также различных врагов с уникальным поведением.

## 🎮 Геймплей

### Управление

| Клавиша | Действие |
|---------|----------|
| **A / D** | Перемещение влево/вправо |
| **Пробел** | Прыжок |
| **Левая кнопка мыши** | Атака мечом |
| **ЛКМ + Shift** | Сильная атака (тратит ману) |
| **Правая кнопка мыши** | Магический огонь (тратит ману) |
| **Shift** | Бег |

### Особенности

- **Два скина персонажа**: на выбор (ведьмак с мечом и ведьмак-мутант)
- **Система комбо-атак**: лёгкие и сильные удары мечом
- **Магия**: огненные шары для атаки на расстоянии
- **Характеристики**: 
  - HP (здоровье) — 16 единиц
  - MP (мана) — 7 единиц
- **Бесконечный раннер**: карта циклически повторяется

## 👾 Враги

| Враг | Описание | HP | Поведение |
|------|----------|-----|-----------|
| **Утопец (Drowner)** | Ближний бой | 10 | Преследует игрока, атакует вблизи |
| **Скелет (Skeleton)** | Ближний бой | 10 | Преследует игрока, атакует вблизи |
| **Маг (Mage)** | Дальний бой | 20 | Держит дистанцию, атакует молниями |
| **Босс (Boss)** | Босс | 30 | Усиленная версия с большей живучестью |

## 🎯 Цель игры

Уничтожайте врагов и набирайте максимальный счёт. Игра заканчивается, когда здоровье ведьмака упадёт до нуля. Рекорд сохраняется в файле `score.txt`.

## 🚀 Установка и запуск

### Требования

- Python 3.8+
- Pygame 2.1+

### Установка

1. Клонируйте репозиторий или скачайте проект
2. Создайте виртуальное окружение (рекомендуется):
   ```bash
   python -m venv venv
   ```

3. Активируйте виртуальное окружение:
   - **Linux/macOS**:
     ```bash
     source venv/bin/activate
     ```
   - **Windows**:
     ```bash
     venv\Scripts\activate
     ```

4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

5. Запустите игру:
   ```bash
   python main.py
   ```

## 🎵 Музыка и звуки

Игра использует музыкальные треки из папки `source/GUI/music/`:
- `menu_music.mp3` — музыка в главном меню
- `game_music.mp3` — музыка во время игры

## ⚙️ Технические детали

- **Разрешение экрана**: 1600×900
- **FPS**: 60
- **Движок**: Pygame 2.x
- **Язык**: Python 3.x

## 📝 Примечания

- Игра использует **пиксельную коллизию** (mask-based) для точного определения столкновений
- Камера следит за игроком и циклически перемещает карту
- Все ресурсы (спрайты, музыка) должны находиться в папке `source/`
